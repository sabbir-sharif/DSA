### Buble sort
- Outer loop - iteration it compare from index 0 to ((n-1)  with next index value
- Inner loop - 0 to ((n - 1) - turn)
- If arr[i] > arr[i+1] then swap (Ascending)
- If arr[i] < arr[i+1] then swap (Descending)

Complexity:
- Time: Worst case: O(n^2)
        Best case: O(n)
- Space: O(n)
